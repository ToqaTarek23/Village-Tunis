# Village Tunis

A beautiful website showcasing the village of Tunis in Fayoum, Egypt.

## About

The village Tunis is located in Egypt and is one of the villages of Youssef Al-Siddiq Centre in Fayoum governorate. The village is beautiful in nature, situated on top of the land with beautiful green farmland underneath, Lake Qarun, and mountain ranges to the north.

## Features

- **Gallery**: Browse beautiful images of the village
- **Activities**: Explore activities available in Tunis Village
- **Contact**: Get in touch with us
- **Responsive Design**: Optimized for all device sizes

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts (Tajawal)

## Getting Started

Simply open `index.html` in your web browser to view the website.
